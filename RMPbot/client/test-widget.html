🤖 RMP Chat Widget

Testing the Respect Me Project chatbot widget integration

Test Instructions:

  1. Look for the blue chat bubble in the bottom-right corner
  2. Click on the chat bubble to open the chat window
  3. Type a message and press Enter or click Send
  4. Wait for the bot response (you'll see typing dots)
  5. Try asking questions like "What is the Respect Me Project?"
Note: Make sure your server is running on localhost:3000 for local testing, or update the serverUrl in the widget configuration for production.
