# 🚀 GET YOUR CHATBOT LIVE IN 15 MINUTES

## The Super Simple Way (100% Free Forever)

### ⚡ QUICK START:

1. **Deploy Backend** (5 min)
   - Go to [Railway.app](https://railway.app) 
   - Login with GitHub
   - "New Project" → "Deploy from GitHub" → Select this repo
   - Railway will auto-detect it's a Node.js app (files are now in root directory)
   - Add environment variable: `GEMINI_API_KEY = your_key`
   - Copy your Railway URL (like: `https://yourapp-production-1234.up.railway.app`)

2. **Add to Wix** (3 min)
   - Open Wix editor → Add "HTML Embed" element
   - Paste: `<script src="https://YOUR-RAILWAY-URL.com/widget.js"></script>`
   - Publish site

3. **Done!** 🎉
   - Blue chat bubble appears bottom-right
   - Click to chat with AI

### 🧪 TEST FIRST:
- Open `RMPbot/client/test-live.html` in browser
- Enter your Railway URL
- Test everything works before adding to Wix

### 📁 IMPORTANT FILES:
